## O que foi feito

### Segurança e Deploy
- Revisão de segurança pré-push: verificado `.gitignore`, ausência de credenciais hardcoded no código, uso correto de `process.env`
- Adicionado `public/uploads/` e `products-import.csv` ao `.gitignore`

- Favicon adicionado via `metadata.icons` no `layout.tsx` apontando para `/logo-unicontrol.svg`

### Logo da empresa (Settings)
- **Problema:** API salvava logo em `public/uploads/` — não funciona na Vercel (filesystem somente leitura)
- **Solução:** Logo convertida para base64 e salva diretamente no campo `logoUrl` da tabela `Company`
- **Bug secundário corrigido:** a `<img>` tinha `src={logoUrl + "?v=0"}` que quebrava data URLs — removido o cache-bust

### Etiquetas de Estoque (`label-print-modal.tsx`)
- Adicionado seletor de tamanho: **Pequena / Média / Grande**
- Substituído download `.docx` por **impressão HTML em nova janela** — mais preciso, usa unidades em `mm`, cabe múltiplas etiquetas por folha
- Corrigido espaço em branco no lado direito: `width: ${mm}mm` → `width: max-content` (borda abraça só o conteúdo)

### Performance do Estoque (`products-tab.tsx`)
- **Problema:** 2137 produtos renderizados de uma vez causavam delay de 2-3s ao mudar de aba
- **Solução:** Paginação client-side de 50 itens por vez
  - Dados completos permanecem em memória (Saída/Entrada ainda têm a lista inteira)
  - Rodapé com "1–50 de 2137 produtos" e botões de navegação
  - Busca reseta para página 1 automaticamente

### Cursor e UX
- Adicionado `button:not(:disabled) { cursor: pointer }` globalmente em `globals.css`
- Loading skeleton melhorado no estoque: imita estrutura real da tabela

## Decisões Técnicas

- **Vercel + VPS:** código no Vercel, banco PostgreSQL na VPS. A porta 5432 estava aberta externamente, o que permitiu a conexão.
- **Base64 para logo:** solução sem serviço externo (sem S3, sem Cloudinary). Funciona bem para uma logo por empresa.
- **Paginação client-side:** preferida sobre server-side paginação pois os dados já estão em memória e os outros tabs (Saída/Entrada) precisam da lista completa.
- **HTML print para etiquetas:** `.docx` descartado — não é adequado para grades de rótulos com dimensões físicas precisas.

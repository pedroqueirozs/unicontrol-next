# Setor de Expedição

## Colaboradores
- 1 colaborador (com possibilidade de aumento)

## Fluxo de Expedição

```
Recebe pedido impresso + endereços (vindo do Administrativo)
      ↓
Arruma os produtos nas caixas
      ↓
Anota dimensões de cada caixa (Largura x Comprimento x Altura e Peso)
      ↓
Passa para o Administrativo fazer cotação do frete
      ↓
Aguarda nota fiscal (emitida pela Contabilidade)
      ↓
Libera a mercadoria quando a transportadora vem buscar
```

## Fluxo de Recebimento de Mercadorias (Fornecedores)

```
Recebe mercadoria do fornecedor
      ↓
Confere a mercadoria com o documento que acompanhou
      ↓
Se tudo certo: assina com data + marca "OK" na nota
      ↓
Se faltou algo: anota o que faltou → Administrativo/Compras resolve com fornecedor
```

## Responsabilidades

- Arrumar pedidos para envio
- Anotar dimensões e peso das caixas (para cotação de frete)
- Conferência de recebimento de mercadorias de fornecedores
- Organização do estoque
- Entrega de encomendas dentro da cidade
- Impressão de etiquetas para caixas ("cuidado frágil", "este lado para cima", etc.)

## Problemas Identificados

### Controle de Estoque Inexistente
- Não há controle de estoque formal
- Verificação de disponibilidade é feita **presencialmente** no estoque
- Após a venda, nenhuma baixa é registrada
- O sistema atual tem produtos cadastrados, mas quando a NF é emitida a baixa não é aproveitada
- Vendas e expedição se comunicam constantemente para verificar disponibilidade
- **Desafio:** cultura organizacional resistente — solução precisa ser muito simples e intuitiva

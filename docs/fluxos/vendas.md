# Setor de Vendas

## Colaboradores
- 2 colaboradores (com possibilidade de aumento)

## Canais de Venda
- Instagram
- WhatsApp
- Telemarketing
- Site

## Fluxo de Venda

```
Realiza a venda
      ↓
Imprime o pedido no sistema (sistema contratado)
      ↓
Passa pedido impresso para o Administrativo
      ↓
Administrativo digita e imprime os endereços (word)
      ↓
Pedido + endereços são enviados para a Expedição
      ↓
Expedição arruma o pedido
```

## Responsabilidades

- Vendas via redes sociais, telemarketing e site
- Impressão dos pedidos no sistema
- Orçamentos de compras para clientes
- Consulta de relatórios de vendas por vendedor (com filtro de período)

## Problema Identificado

- O setor de vendas realiza o **cadastramento de novos produtos** no sistema, porém isso **deveria ser feito pelo contador**, devido a particularidades fiscais no cadastro dos produtos.
